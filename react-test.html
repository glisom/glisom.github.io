React Test Project

This test is to see how well you can adapt to a new framework, in this case, React Native. The project should not take very long, and we will be available as you progress with any advice or direction you need. The final deliverable should be a React Native app (optimized for iOS or Android, preferably iOS) that contains a list view screen with items that each have a picture, a text field, a radio button (something that can be toggled) and some indicator that the user can press that item and move to a different screen. The user should be able to add and remove items from the list. When a user presses an item in the list (other than the toggled element), a more detailed view should be shown containing more interactive items (use creativity on this one).
