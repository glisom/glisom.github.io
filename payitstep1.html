Ionic Project

For this project, you are to implement an Ionic Framework webapp using either AngularJS or a web framework of your choice.

You can simply just build the blank starter app from the Ionic documentation, then using Twitter's API you need to display your's or another public users past 10 tweets. If you want to display more using pagination or something thats cool too, but it must be at least the last ten.

Get Started
