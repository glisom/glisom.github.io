Hi! Welcome to Step 4!

Thank you for taking time to complete this step. This helps us a ton in finding a job that fits your skill set and shows us what you prefer when tackling a small project.

So below you find two small projects, unless you were specifically told to do the second one please only do the first one.

Project 1

About me page. Your task is to build an about me page using either Facebook's API or Twitter's API. Keep it as simple or complex as you want, this just needs to display your name, eamil, picture, and any other information you decide you want to grab from these sites. Remeber looks do matter, so go ahead and make them look professional.

Please use Backbone, Ember, Angular, Knockout, or Reactjs framwork to complete the website. It does not need to be hosted anywhere or anything special, should just work when ran local. Please use Github to track your progress along the way, and when you are finished just send us the link! Good luck and we are looking forward to seeing your project!

Project 2

Song and Information App. Using Spotify's iOS SDK, create an app that plays a song and displays the information about that song. You can hard code the song, but you cannot hard code the information about that song.

Please feel free to use either Swift or Objective-C, and send us a link to your Github project when it is completed. Make sure to commit a few times throughout the process so we can see your work progress. Good luck and looking forward to seeing it!
